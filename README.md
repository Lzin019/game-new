## 🎯 Sobre o Projeto
O **Clube do Livro** é uma plataforma digital colaborativa desenvolvida em parceria entre o **SENAI** (Desenvolvimento de Sistemas) e o **SESI** (Língua Portuguesa e Inglês). 

O objetivo central é preparar alunos para vestibulares através de uma ferramenta interativa e multilíngue, focada no estudo aprofundado de obras literárias clássicas e contemporâneas.

---

## 🚀 Tecnologias Utilizadas

| Camada | Tecnologia | Descrição |
| :--- | :--- | :--- |
| **Backend** | `Node.js` | API REST para gerenciamento de dados. |
| **Frontend** | `React` / `React Native` | Interface Web ou Mobile responsiva. |
| **Banco de Dados** | `PostgreSQL` | Armazenamento estruturado e relacional. |
| **Documentação** | `Postman` / `Swagger` | Testes e especificação da API. |
| **Versionamento** | `Git` | Controle de versão obrigatório. |

---

## 🌍 Diferencial Multilíngue (SESI)
O sistema foi projetado para ser totalmente acessível em dois idiomas:
- 🇧🇷 **Português** (Padrão)
- 🇺🇸 **Inglês** (Tradução humana e adaptada)

> **Nota:** Todo o conteúdo foi adaptado sem o uso de ferramentas de tradução automática, visando a prática real da língua inglesa.

---

## 🛠️ Funcionalidades Principais

### 🏠 Áreas do Sistema
- **Home:** Destaque para o livro principal e visão geral.
- **Sobre:** Informações sobre a equipe e objetivos acadêmicos.
- **Análise Literária:** Resumo, personagens e contexto histórico da obra principal.
- **📚 Biblioteca Integrada:** Listagem de livros de todas as equipes através de consumo de APIs externas.
- **🎓 Preparatório:** Simulados, Quizzes estilo vestibular e dicas de redação.
- **🎥 Conteúdo Multimídia:** Videoaulas autorais explicativas.
- **💡 Curiosidades:** Fatos marcantes sobre autores e época.

## 📂 Estrutura de Entregáveis

O projeto segue um rigoroso padrão de documentação e desenvolvimento:

1.  **Documentação Técnica:**
    * Escopo, Requisitos e Regras de Negócio.
    * Cronograma de Gantt e Gestão Kanban.
    * Diagrama de Entidade-Relacionamento (DER).
2.  **Desenvolvimento:**
    * CRUD Completo.
    * Consumo de rotas `GET` e `GetById`.
    * Deploy do Backend.

---

## ⚠️ Regras de Desenvolvimento
Para garantir a integridade acadêmica, o projeto segue as seguintes normas:
- **Stack Fixa:** Apenas tecnologias trabalhadas em sala de aula.
- **Código Autoral:** Proibido o uso de IA para geração de código e plágio.
- **Performance:** Tempo de resposta inferior a 4 segundos.
